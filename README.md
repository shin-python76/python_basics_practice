## 📝 学習メモ
- `input()` はユーザー入力を文字列として受け取る  
- `int()` や `float()` を使うと文字列を数値に変換できる  
- `if` 文の条件分岐では、複数の条件を順序に注意して書く  
- `sum()` と `len()` を組み合わせてリストの平均を求める  
- `dict.items()` で辞書のキーと値を同時に取り出せる  
- `\n` は改行を意味するエスケープシーケンス  
- `return` は関数の計算結果を呼び出し元に返す
